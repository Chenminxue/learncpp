### Comments
Single-line comments

```markdown
 // ...
```

Multi-line comments

```markdown
 /* ...
    ...
    ... */
```

### Variables
Give a name to a space in memory.

```markdown
 type variable_Name = value;

 int a = 10;
```

### Constants
Unchangeable data in the program.

```markdown
 #define Constant_Name value
 
 #define DAY 7
```

```markdown

 const Variable_Name = value;

 const int month = 12;
```
